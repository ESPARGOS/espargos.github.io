ESPARGOS Web Tunnel

Connect to an ESPARGOS device over USB and use its built-in web interface right here in your browser, via Web Serial API. Ideal for the initial network configuration setup!

Experimental feature: The Web Tunnel is highly experimental and may not work with all operating systems, browsers and computers. If you run into issues, access the web interface over the recommended Ethernet network connection instead.
Your browser does not support the Web Serial API required for the Web Tunnel. Please use a recent desktop browser that supports it.
